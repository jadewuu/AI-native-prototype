# PM 需求 Prompt 模板

> 使用说明：复制下方模板，填写 [] 中的内容，发送给 AI 即可。
> 不确定的字段可以填"待定"，AI 会生成默认值并说明。

---

## 模板 A：列表页

```
请帮我生成一个列表页，需求如下：

【页面名称】[例：产品管理]
【页面用途】[一句话说明这个页面是干什么的，例：管理所有产品，支持搜索、查看和删除]

【表格字段】
- [字段名]：[字段说明，例：产品名称：文字，最长 50 字]
- [字段名]：[例：分类：标签，可选值 Electronics / Clothing / Food]
- [字段名]：[例：价格：数字，格式 $X.XX]
- [字段名]：[例：创建时间：日期时间]

【操作】
- 主操作：[例：点击 Manage 跳转详情页]
- 次要操作：[例：复制、删除]
- 新建入口：[例：右上角 + Add 按钮，点击打开新建 Dialog]

【搜索/筛选】
- 搜索：[例：按产品名称搜索 / 无]
- 筛选：[例：按分类筛选 / 无]

【其他说明】[例：表格默认按创建时间倒序排列 / 无]
```

---

## 模板 B：仪表板页

```
请帮我生成一个数据仪表板页，需求如下：

【页面名称】[例：运营 Dashboard]
【页面用途】[例：展示核心业务数据概览，支持按时间和分类筛选]

【顶部筛选条件】
- [例：日期范围选择器]
- [例：分类下拉筛选，选项：All Category / 各分类名称]
- [例：渠道下拉筛选，选项：All Channel / 各渠道名称]

【指标卡片】（最多 4 个）
- [例：Revenue / 总收入 / 与昨日对比]
- [例：Orders / 订单数 / 与昨日对比]
- [例：Users / 新增用户 / 与昨日对比]
- [例：Conversion / 转化率（%）/ 与昨日对比]

【图表】
- 图表1（左大）：[例：营收趋势柱状图，维度：按日/周/月]
- 图表2（右小）：[例：订单来源分布饼图，维度：各渠道占比]
- 图表3（左大）：[例：用户增长趋势折线图]
- 图表4（右小）：[例：分类销售额柱状图，维度：各分类]

【其他说明】[例：图表上方需要展示文字统计数据 / 无]
```

## 模板 C：表单页

```
请帮我生成一个表单页，需求如下：

【页面名称】[例：新建产品]
【页面用途】[例：填写产品信息并提交创建]
【页面类型】[新建 / 编辑]

【表单字段】
- [字段名]：[字段说明，例：产品名称：文本输入，必填，最长 50 字]
- [字段名]：[例：分类：下拉选择，必填，选项为 Electronics / Clothing / Food]
- [字段名]：[例：价格：数字输入，必填，格式 $X.XX]
- [字段名]：[例：描述：富文本，选填]
- [字段名]：[例：上架状态：开关，默认开启]
- [字段名]：[例：生效日期：日期选择器，必填]

【表单分组】
[例：基本信息（名称、分类、描述）、价格信息（价格、货币）、状态设置（上架状态、生效日期）]

【提交行为】
- 提交按钮文字：[例：Save / Create]
- 成功后：[例：跳转回列表页 / 留在本页]
- 失败后：[例：Toast 提示错误信息]

【其他说明】[例：编辑模式下字段预填已有数据 / 无]
```

## 模板 D：详情页

```
请帮我生成一个详情页，需求如下：

【页面名称】[例：产品详情]
【页面用途】[例：查看产品完整信息，支持编辑和删除]
【所属资源】[例：产品 / 订单 / 用户]

【信息展示】
- 基本信息卡片：[例：名称、分类、价格、描述、状态标签]
- 元数据卡片：[例：创建人、创建时间、最后修改时间]
- 统计卡片（可选）：[例：总销量、总浏览量]

【操作按钮】（页面右上角）
- [例：Edit 按钮，跳转编辑页]
- [例：Delete 按钮，弹出确认 Dialog]

【Tab 切换】（如有）
- Tab 1：[例：订单记录 — 关联的订单列表]
- Tab 2：[例：操作日志 — 时间线展示]

【其他说明】[例：状态标签 Active 用绿色，Inactive 用灰色 / 无]
```

## 常用补充指令

生成之后，如果需要调整，可以直接发：

```
把表格的 [列名] 列去掉

在 [列名] 列加一个状态标签，颜色：Active 用绿色，Paused 用灰色，Stopped 用红色

搜索框改成支持按 [字段A] 和 [字段B] 搜索

新建操作改成跳转新页面，不用 Dialog

图表 [图表名] 改成折线图

在表单中增加 [字段名] 字段，类型为 [文本/下拉/日期...]

详情页增加一个 [说明] 的 Tab
```

---

## 注意事项

- AI 会严格按照设计规范生成，字体（Nunito Sans）、颜色、布局均固定
- 如果需要突破规范（如改变侧边栏宽度、换字体），需要先确认
- 生成结果如有明显偏差，请直接描述问题，AI 会基于规范重新调整
